## 0. Codex 执行总约束

### 0.1 禁止事项

1. **不要修改任务 V3 业务规则。**  
   任务 V3 只允许在 P0-07 E2E 中调用和校验，不允许改随机商品、用户额度、客服追加商品、奖励结算逻辑。

2. **不要绕过 `WalletLedgerService` 改余额。**  
   禁止直接：
   ```python
   wallet.system_balance += amount
   wallet.system_balance -= amount
   wallet.task_balance += amount
   wallet.task_balance -= amount
   ```

3. **不要在支付、提现、任务、Webhook 中写隐式副作用。**  
   所有副作用必须：
   - 有明确 service 方法
   - 有 idempotency key
   - 有 ledger / audit
   - 有测试

4. **不要让 mock、默认 secret、默认 admin、localhost CORS 进入 production。**

5. **不要删除错误流水。**  
   错误资金操作只能做 reverse / correction ledger。

6. **不要只改前端。**  
   P0 项目都必须有后端校验、测试和上线脚本。

```python
from __future__ import annotations

from dataclasses import dataclass
from urllib.parse import urlparse

from app.core.settings import Settings


class ProductionGuardError(RuntimeError):
    pass


@dataclass(slots=True)
class ProductionGuardIssue:
    code: str
    message: str
    severity: str = "S"


UNSAFE_ADMIN_SECRETS = {
    "",
    "change-me-in-production",
    "changeme",
    "secret",
    "admin",
}

UNSAFE_PASSWORDS = {
    "",
    "admin",
    "admin123",
    "password",
    "123456",
}


def collect_production_issues(settings: Settings) -> list[ProductionGuardIssue]:
    issues: list[ProductionGuardIssue] = []
    env = (settings.app_env or "").strip().lower()
    if env != "production":
        return issues

    if settings.admin_jwt_secret.strip() in UNSAFE_ADMIN_SECRETS or len(settings.admin_jwt_secret.strip()) < 32:
        issues.append(ProductionGuardIssue(
            code="UNSAFE_ADMIN_JWT_SECRET",
            message="ADMIN_JWT_SECRET is empty/default/too short.",
        ))

    if (settings.admin_default_username or "").strip().lower() == "admin":
        issues.append(ProductionGuardIssue(
            code="DEFAULT_ADMIN_USERNAME_ENABLED",
            message="ADMIN_DEFAULT_USERNAME must not be 'admin' in production.",
        ))

    if (settings.admin_default_password or "").strip() in UNSAFE_PASSWORDS:
        issues.append(ProductionGuardIssue(
            code="DEFAULT_ADMIN_PASSWORD_ENABLED",
            message="ADMIN_DEFAULT_PASSWORD is default/empty in production.",
        ))

    if (settings.messaging_provider or "").strip().lower() == "mock":
        issues.append(ProductionGuardIssue(
            code="MOCK_MESSAGING_PROVIDER",
            message="MESSAGING_PROVIDER=mock is not allowed in production.",
        ))

    if (settings.ai_provider or "").strip().lower() == "mock":
        issues.append(ProductionGuardIssue(
            code="MOCK_AI_PROVIDER",
            message="AI_PROVIDER=mock is not allowed in production.",
            severity="A",
        ))

    if not settings.webhook_signature_enabled:
        issues.append(ProductionGuardIssue(
            code="WEBHOOK_SIGNATURE_DISABLED",
            message="WEBHOOK_SIGNATURE_ENABLED must be true in production.",
        ))

    if not settings.h5_member_cookie_secure:
        issues.append(ProductionGuardIssue(
            code="H5_COOKIE_INSECURE",
            message="H5_MEMBER_COOKIE_SECURE must be true in production.",
        ))

    if settings.h5_member_cookie_samesite.lower() not in {"lax", "strict", "none"}:
        issues.append(ProductionGuardIssue(
            code="INVALID_COOKIE_SAMESITE",
            message="H5_MEMBER_COOKIE_SAMESITE must be lax/strict/none.",
        ))

    cors_origins = [item.strip() for item in settings.cors_origins.split(",") if item.strip()]
    if not cors_origins:
        issues.append(ProductionGuardIssue(
            code="EMPTY_CORS_ORIGINS",
            message="CORS_ORIGINS must be configured in production.",
        ))
    for origin in cors_origins:
        parsed = urlparse(origin)
        if origin == "*" or parsed.hostname in {"localhost", "127.0.0.1", "0.0.0.0"}:
            issues.append(ProductionGuardIssue(
                code="UNSAFE_CORS_ORIGIN",
                message=f"Unsafe production CORS origin: {origin}",
            ))
        if parsed.scheme != "https":
            issues.append(ProductionGuardIssue(
                code="NON_HTTPS_CORS_ORIGIN",
                message=f"Production CORS origin must be https: {origin}",
                severity="A",
            ))

    if (settings.meta_management_provider or "").strip().lower() == "whatsapp":
        if not settings.meta_app_secret.strip():
            issues.append(ProductionGuardIssue(
                code="MISSING_META_APP_SECRET",
                message="META_APP_SECRET is required when META_MANAGEMENT_PROVIDER=whatsapp.",
            ))
        if not settings.meta_global_webhook_verify_token.strip():
            issues.append(ProductionGuardIssue(
                code="MISSING_META_VERIFY_TOKEN",
                message="META_GLOBAL_WEBHOOK_VERIFY_TOKEN is required in production.",
            ))

    return issues


def assert_production_safe(settings: Settings) -> None:
    issues = collect_production_issues(settings)
    blocking = [issue for issue in issues if issue.severity == "S"]
    if blocking:
        detail = "; ".join(f"{item.code}: {item.message}" for item in blocking)
        raise ProductionGuardError(f"Production guard failed: {detail}")
```

### 7.1 新增 `app/services/wallet_invariant_guard.py`

```python
from __future__ import annotations

from dataclasses import dataclass
from decimal import Decimal

from app.db.models import WalletAccount, WalletLedgerEntry


TWOPLACES = Decimal("0.01")


class WalletInvariantError(RuntimeError):
    pass


@dataclass(slots=True)
class WalletInvariantIssue:
    code: str
    message: str


class WalletInvariantGuard:
    @staticmethod
    def q(value) -> Decimal:
        return Decimal(value or 0).quantize(TWOPLACES)

    @classmethod
    def validate_wallet(cls, wallet: WalletAccount) -> None:
        issues: list[WalletInvariantIssue] = []

        system_balance = cls.q(wallet.system_balance)
        cash = cls.q(wallet.system_cash_balance)
        bonus = cls.q(wallet.system_bonus_balance)
        frozen = cls.q(wallet.frozen_balance)
        cash_frozen = cls.q(wallet.system_cash_frozen)
        bonus_frozen = cls.q(wallet.system_bonus_frozen)
        task = cls.q(wallet.task_balance)

        if system_balance != cls.q(cash + bonus):
            issues.append(WalletInvariantIssue(
                "SYSTEM_BALANCE_MISMATCH",
                f"system_balance={system_balance}, cash+bonus={cash + bonus}",
            ))

        if frozen != cls.q(cash_frozen + bonus_frozen):
            issues.append(WalletInvariantIssue(
                "FROZEN_BALANCE_MISMATCH",
                f"frozen_balance={frozen}, cash_frozen+bonus_frozen={cash_frozen + bonus_frozen}",
            ))

        for name, value in {
            "system_balance": system_balance,
            "system_cash_balance": cash,
            "system_bonus_balance": bonus,
            "frozen_balance": frozen,
            "system_cash_frozen": cash_frozen,
            "system_bonus_frozen": bonus_frozen,
            "task_balance": task,
        }.items():
            if value < 0:
                issues.append(WalletInvariantIssue("NEGATIVE_BALANCE", f"{name}={value}"))

        if issues:
            detail = "; ".join(f"{item.code}: {item.message}" for item in issues)
            raise WalletInvariantError(detail)

    @classmethod
    def validate_ledger_entry(cls, entry: WalletLedgerEntry) -> None:
        amount = cls.q(entry.amount)
        cash = cls.q(getattr(entry, "cash_amount", 0))
        bonus = cls.q(getattr(entry, "bonus_amount", 0))
        task = cls.q(getattr(entry, "task_amount", 0))
        ledger_type = getattr(entry, "ledger_type", None)

        if amount < 0:
            raise WalletInvariantError("LEDGER_AMOUNT_NEGATIVE")

        if ledger_type == "system":
            if cls.q(cash + bonus) != amount:
                raise WalletInvariantError(
                    f"LEDGER_SPLIT_MISMATCH: amount={amount}, cash+bonus={cash + bonus}"
                )

        if ledger_type == "task":
            if task not in {Decimal("0.00"), amount}:
                raise WalletInvariantError(
                    f"TASK_LEDGER_AMOUNT_MISMATCH: amount={amount}, task_amount={task}"
                )

        if not getattr(entry, "source_type", None):
            raise WalletInvariantError("LEDGER_SOURCE_TYPE_REQUIRED")

        if not getattr(entry, "reference_type", None) or not getattr(entry, "reference_id", None):
            raise WalletInvariantError("LEDGER_REFERENCE_REQUIRED")
```

处理流程：

```text
1. 读取 channel + provider
2. 记录 PaymentCallback(raw_payload)
3. 验签失败：保存失败，不入账
4. normalize payload
5. 校验 amount > 0
6. 校验 currency
7. 校验 user_id 存在
8. 生成三重幂等 key：
   - payment_callback:{callback_id}
   - payment_channel_order:{channel_id}:{channel_order_no}
   - payment_platform_order:{channel_id}:{platform_order_no}
9. 用数据库锁锁定 channel_order_no 对应入账防重
10. 如果已入账：callback processed_duplicate
11. 如果未入账：
    - 创建/更新 RechargeRecord
    - 创建 WalletRechargeOrder
    - 调 WalletLedgerService.credit_system_balance(... idempotency_key=payment_channel_order...)
    - 标记 callback processed
12. 真实充值成功后调用 MemberCertificationService.refresh_user_certification()
13. commit
```

# 23. 最终 Definition of Done

完成本文后必须满足：

1. 生产默认配置不安全时应用无法启动。
2. CI 会跑到 `tests/api` 和 `tests/services`。
3. deploy 不会因为缺 `pytest-timeout` 或跳过测试而假通过。
4. 所有钱包写操作都有 idempotency key。
5. 钱包 invariant guard 生效。
6. 日终对账能发现手工篡改余额。
7. 重复充值回调不重复入账。
8. 回调和补单并发不重复入账。
9. 支付对账能识别 missing_platform、missing_channel、amount_mismatch。
10. H5 提现不能重复提交 active withdrawal。
11. 提现有单笔、日/周/月、频率风控。
12. 打款失败不会退钱，取消/驳回才退钱。
13. 打款成功只释放 frozen 一次。
14. 生产 E2E 跑通完整链路。
15. 任务 V3 核心逻辑未被 P0 剩余模块改坏。
16. 后端 pytest、前端 typecheck/build/test 全部通过。
