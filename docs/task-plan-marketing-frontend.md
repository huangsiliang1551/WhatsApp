## 四、CustomersPage 增强设计

### 新增筛选条件

在现有筛选基础上追加：

| 筛选 | 类型 | 说明 |
|------|------|------|
| 签到状态 | Select | 已签到/未签到/连续N天以上 |
| 任务状态 | Select | 有进行中任务/无任务/已完成任务 |
| 邀请数 | 范围 | 邀请人数 ≥ N |
| 充值金额 | 范围 | 累计充值 ≥ N |
| 标签 | MultiSelect | 用户标签筛选 |

### 行操作增强

| 操作 | 说明 |
|------|------|
| 推送任务 | 点击 → 弹出 ManualPushModal（选商品包 → 确认） |
| 详情 | 打开 Drawer，新增任务/签到/邀请区块 |
