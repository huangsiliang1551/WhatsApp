# DECISIONS

## 已确认业务规则

- 任务 V3 业务规则冻结，P0剩余模块只在 E2E 调用任务 V3。
- WhatsApp 采用站点号码池共享服务号模式。
- 每个 H5 站点可配置多个 WABA / Phone Number。
- 一个 `phone_number_id` 同一时间只能属于一个 active 站点池；站点关闭后可释放并迁移。
- `wa_id` 全平台唯一绑定。
- H5A绑定后，H5B使用同一 `wa_id` 注册/登录/绑定必须失败。
- 未绑定普通消息不进入正式 AI，只回复绑定引导。
- 已绑定用户发到同站点号码池其他号码，合并同一客户会话。
- 号码低质量时停止新用户分配，继续服务老客户主动咨询。
- 号码失效不解除绑定，后续 H5 链接迁移到健康号码。
- 权限架构为：超级管理员 → 代理商 → 主管 → 成员。
- 数据范围也必须漏斗式收窄。
- 会话临时转接和客户永久归属转移必须分开。
- B服务器只跑H5网关/静态前端/Agent，不跑A后端。
